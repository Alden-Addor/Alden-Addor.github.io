PROJECTS



G-Kamp
G-Kampclose

G-Kamp is a camping specific solution to Recreation.gov. It uses the government API to filter out campgrounds only from REC.gov. The project uses Python, Django, Vue, JavaScript, HTML, CSS, Materialize (as a styling framework), SQLite as the database, Ajax, and Leaflet for mapping. This project used session manipulation also for saving data between views and the different API calls that were required. Currently, it is still just on my Github page, but I am in the process of including more features and deploying as a live website.

Groundedlocal_bar
Groundedclose

Grounded is a drink site that allows you to look up alcoholic and non-alcoholic drink ideas and recipes. It uses a third-party API thecocktaildb for the data. Vue and Ajax were the main focus of the project, with HTML, CSS, and Materialize filling out the rest of the programming. The link and this project is currently the assignment version from my coding boot camp. In the future, I intend to add functionality and create a more user-friendly version that I will deploy live.
